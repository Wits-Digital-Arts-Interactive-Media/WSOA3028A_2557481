JavaScript

The navigation.js code defines an array named navLinks containing objects representing navigation links for my website. The 'createNavigationBar()' function dynamically generates a navigation bar based on the navLinks array, creating list items and anchor elements for each link. These elements are then appended to a 'nav' element in the HTML document. Finally, the function is called when the DOM content is fully loaded, ensuring the navigation bar is created dynamically after the HTML content is parsed.

The colour-palette.js code offers functionality to dynamically change and reset the background colour of my webpage. Initially, it defines a constant originalColor storing the original background colour. Two functions, changeBackgroundColour() and resetBackgroundColour(), are then introduced. The former selects a random colour from an array of predefined colour values and sets it as the background colour of the document body. Conversely, the latter restores the original background colour by assigning the originalColour value to the document body's background colour style property. This code enables the implementation of an interactive feature where users can experience a variety of background colours by clicking a button, while also having the option to revert to the default background colour.

The quote.js code defines an array called quotes, containing several famous quotes from iconic movies. The generateQuote() function is created to randomly select one of these quotes and display it on my webpage. Within the function, a random index is generated based on the length of the quotes array, ensuring that it falls within the bounds of the array's indices. The selected quote is then retrieved from the quotes array using the random index and assigned as the text content of an HTML element with the ID "quote", effectively updating the displayed quote each time the function is invoked. This code snippet enables the creation of a simple quote generator feature that adds an element of randomness and entertainment to my webpage, potentially enhancing user engagement and interaction.
